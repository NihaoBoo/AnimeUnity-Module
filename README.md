# AnimeUnity (ITA) - Synthetiq Module

Questo è un modulo video per **Synthetiq Player** che permette l'estrazione e lo streaming dal catalogo di AnimeUnity.

## Come Importare in Synthetiq

Puoi installare questo modulo direttamente nell'app fornendo il link "Raw" del `manifest.json`.

1. Apri **Synthetiq Player** e vai nelle impostazioni dei Moduli.
2. Clicca su **Aggiungi Modulo da URL**.
3. Incolla il link "Raw" del file `manifest.json` di questa repository:
   `https://raw.githubusercontent.com/NihaoBoo/AnimeUnity-Module-for-MediaPlayer-Synthetiq-/main/manifest.json`
4. Conferma per installare.

## Funzionalità
- Ricerca nel catalogo
- Dettagli Anime e lista Episodi
- Estrazione automatica e bypass per Vixcloud (.m3u8)
